Reading Files

  • open — Open a file.
  • close — Closes the file.Like File->Save..in yout editor.
  • read — Reads the contents if the file.You can assign the result to a variable.
  • readline — Reads just one of a text file.
  • truncate — Empties the file.Watch out if you care about the file.
  • write('stuff') — Writes “stuff” to the file.
